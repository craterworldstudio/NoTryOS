A quick guide on how to compile/RUn your asm file.

Head over to your Projects directory,
I will head to leanable_programs

cd leanable_programs

run nasm:
nasm -f elf64 YourProgram.asm

this will create a instruction set called YourProgram.o (an object file) for the cpu to follow and execute.
Next we need to create a executable from this:
ld YourProgram.o -o YourProgram

Run your program with,
./YourProgram

If you want to see the exit code of the programs
just execute,

echo $?





Later when we write the bootloader, we will ditch the elf64 header altogether and stick to -f bin which is raw binary.

Also Install qemu-base for the bootloader and the os to run as a whole.
