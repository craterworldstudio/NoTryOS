'''
global _start

section .data
password dd 123456
guess dd 837377 ; initial guess

section .text

success:
  mov rax, 60
  mov rdi, 0
  syscall

failure: 
  mov rax, 60
  mov rdi, 1
  syscall

_start:
  mov eax, [password]
  push rax
  mov ecx, [guess]
  pop rbx

  cmp rcx, rbx
  je success
  jne failure
'''

At first we define the _start, the entrypoint of our program.

next we define the data section and set password and guess as the two hardcoded inputs for now as doubleword data
in the text section, first we create 2 methods. success or failure with exit code 0 and 1 respectively (usually whats used in most cases.)

in the start method, we move the password from a local memory to a fast register of same length.
Next to prevent the password being modified accidentally, we push it to stack to store it safely while still having fast access.

next we take the guessed password from local memory to a register and simultaneosly, pop our actual password out.

next we compare the two. If they match or gets 'equal', then jump to success, or else failure.
